whiteBroad

Pizarra virtual

Whiteboard es una aplicación de pizarra virtual muy útil para las clases virtuales, sin embargo, de no contar con una pantalla táctil, al escribir se debe utilizar el mouse, esto no suele ser muy práctico, para esto se explica cómo configurar un celular con sistema operativo IoS para poder usar su pantalla táctil para escribir y transmitir este mensaje a la computadora.

Como usar WhiteBoard si tu no tienes una PC tactil

LiveBoard

Pizarra virtual

LiveBoard es una pizarra online que puede ser utilizada desde un navegador o mediante un dispositivo móvil. En este tutorial se explican diferentes aspectos como: registrar una cuenta, instalar la aplicación en un dispositivo Android y la forma de transmitir la pantalla del celular hacia la del navegador en la PC. Esta propiedad ayuda a mostrar información en una clase virtual con la opción de compartir pantalla en cualquiera de las plataformas (Zoom, Meet, Teams).

Kahoot

Lorem ipsum dolor sit amet consectetur.

Kahoot es una aplicación utilizada para hacer preguntas de forma interactiva, esta aplicación ayuda a crear competitividad, dando así una mejora en el estilo de las clases. En este tutorial se explican los beneficios de la aplicación y la forma de utilizarla para crear diferentes tipos de preguntas interactivas.

Zoom

Plataforma de reuniones.

Zoom es una aplicación de conferencias muy utilizada en las clases virtuales, en este tutorial se explica cómo utilizar la cuenta Zoom desde el navegador, además de cómo empezar o unirse a una video llamada desde la aplicación. En adición se muestran las diferentes opciones que ofrece esta plataforma como el whiteboard, pantalla compartida, configuración de audio, video y las diferentes formas de invitar a los contactos a unirse a la reunión.

¿Cómo crear una cuenta de Zoom desde el celular?

En este tutorial se explica desde cero como se instala Zoom en un celular con sistema operativo Android, luego se detalla los procesos a seguir para crear una cuenta Zoom mediante un correo electrónico y así poder crear conferencias.

Google Meet

Plataforma de reuniones.

Meet es la nueva aplicación de videoconferencias de Google que reemplaza a Hangouts. En este tutorial se explica cómo ingresar o crear reuniones mediante Google Meet, además se explica los diferentes funcionamientos de la aplicación como compartir pantalla, subtítulos, configuración de audio y video, entre otros.
